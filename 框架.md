        {
            "isdir": false,
            "name": "001.jpg",
            "size": "292.48KB",

9.文件查找

| 请求方式 | 请求参数 | 返回参数 |
| -------- | -------- | -------- |
| POST     | path     | main     |

示例：

```
http://localhost:8080/file/SearchFile
```

请求参数：

```
{
	"path":"D:/xx",
	"keyword":"main",
	"type":"AtPresent"
}
```

> 说明：`path`为文件查找的路径
>
> ​	`keyword`为查找文件的关键词
>
> ​	`type`为查找种类，分别为`AtPresent`（在当前文件夹下查找，不会在当前文件夹下的文件夹内查找）,`All`  （如果当前的path有内容，就会在当前文件夹下地毯式搜索。如果path为空，则会分别在服务器的各个磁盘内搜索）

返回参数：

```
{
    "KeyWord": "main",// 关键词
    "Number": 2,// 查找的数量
    "Path": "D:/xx/",// 查找路径
    "Type": "AtPresent",// 查找类型
    "main": [
        {
            "isdir": false,
            "name": "NB-Group-main.zip",
            "size": "1.31KB",
            "time": "10月06日",
            "path": "D:/xx/NB-Group-main.zip"
        },
        {
            "isdir": false,
            "name": "main.go",
            "size": "14.21KB",
            "time": "08月18日",
            "path": "D:/xx/main.go"
        }
    ]
}
```
