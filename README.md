## 服务器文件管理系统 (开发中，目前仅支持win server)

#### 使用编程语言及框架
1. 编程语言：goland
2. 开发框架：
   - gin(作为服务端api)
   - os
   - 及其他
  
#### 优点 （只是想象，希望可以做到）
- 更好的管理服务器文件
- 管理服务器配置
- 查看服务器状态(cpu,硬盘，内存)
- 更好的管理多台服务器
- 多台服务器中进行通信，避免跨服务器传输文件
- 文件在线预览（可自定义office文件预览）
- 文件在线编辑（如在编辑中有特殊情况可恢复）
- 文佳下载
- 分享文件（可设置下载次数，到期时间，密码）

<font color=red>服务器之间的文件传输速度和下载速度取决于服务器本身的带宽！！</font>
